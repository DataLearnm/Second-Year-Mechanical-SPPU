# **Data Science & AI-ML Practicals**
### **Savitribai Phule Pune University (SPPU) – Second Year Mechanical Engineering (2024 Pattern)**

### Overview
This repository contains 8 practical assignments designed for Data Science & AI-ML subject as per the curriculum of Second Year Mechanical Engineering (2024 Pattern).

The aim of these practicals is to provide hands-on experience in:
- Machine Learning techniques
- Data analysis & visualization
- Feature engineering
- Model building and evaluation
- Real-world engineering applications

### Learning Objectives
Understand Supervised, Unsupervised & Reinforcement Learning
Work with real-world datasets
Perform data preprocessing & visualization
Build and evaluate ML models
Apply AI/ML concepts to engineering problems

### Practicals List
#### 1. Machine Learning Approaches
- Study:
  - Supervised Learning
  - Unsupervised Learning
  - Reinforcement Learning
- Application:
    - Email Spam Classification
 
#### 2. Data Acquisition & Visualization
- Work with:
  - Time-domain data
  - Frequency-domain data
- Application:
    - Comparison of Engine Vibration Frequencies
 
#### 3. Feature Extraction
- Extract meaningful features from datasets
- Application:
  - 📰 Word Count Extraction from News Articles
 
#### 4. Handwriting Recognition
- Build a model to recognize handwritten digits/characters
- Application:
   - ✍️ Optical Character Recognition (OCR) basics
 
#### 5. Classification Model
- Develop classification models
- Evaluate performance using metrics
- Application:
  - 🌸 Classify Flower Species (e.g., Iris Dataset)
 
#### 6. Regression Model
- Build regression models
- Evaluate predictions
- Application:
  - 🏠 House Price Prediction

#### 7. Markov Process
- Study stochastic processes
- Application:
  - 🏭 Machine Maintenance Modeling
 
#### 8. Reinforcement Learning
- Apply RL concepts in engineering
- Applications:
  - ⚡ Optimize Energy Consumption
  - 🤖 Robot Guidance & Navigation
 
### 🛠️ Technologies Used
- Python 🐍
- NumPy
- Pandas
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook

### 📊 Workflow Followed
Each practical follows a structured approach:
1. Problem Definition
2. Data Collection
3. Data Preprocessing
4. Data Visualization
5. Feature Engineering
6. Model Building
7. Evaluation
8. Conclusion
